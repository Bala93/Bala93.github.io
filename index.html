A platform for Joint Optic Disc and Cup Segmentation for Analysis of Glaucoma

Glaucoma is one of the serious, widespread ocular diseases which may lead to permanent loss in vision. One of the important parameters used for Glaucoma screening in the cup-to-disc ratio (CDR), which requires accurate segmentation of optic cup and disc.
This work involved developing a tool for analysis of Glaucoma. We leveraged our previously published novel work, which used Generative Adversarial Networks (GANs) based Convolutional Neural Network for semantic segmentation of Retinal fundus images. The work outperformed other state of the art works on joint disc and cup segmentation. We built a tool which takes in a fundus image and automatically selects the region of interest using morphological techniques and then assigns pixel-wise labels for the cropped image for 3 classes- cup, disc, background. Using the cup to disc ratio information, the presence or absence of glaucoma is then calculated.

Classify the product images in supermarket into common categories

The largest retailer in Germany wanted to improve their inventory-management process in its Food and Groceries business.   The company needed intelligent solutions that can reduce the amount of human effort in its warehouse and retail outlets. A solution such as powerful image classifier can help the company track shelf inventory, categorize products, record product volume etc.
Hackerearth hosted a challenge to bring out the best possible classifier. The task is to predict the category of the product based on the given image. The dataset provided had train data of 3215 and test data of 1732 product images with a number of categories to 25.
Competing with top data enthusiasts from all over the world (close to 350) participated in the challenge, our team were able to get an accuracy of 90.35% which gave us a spot in top 5%.
Powerful deep neural networks like DenseNet and ResNet were used. As the datasets were not balanced well, data augmentation played a critical role in higher accuracy. Rather than relying on a single model an ensemble of top networks is used which gave us a boost in performance.

Face recognition -- A tool to identify or verify a person from a given image

Face recognition has major application in many fields. Lynk is a company which helps to connect the customer with truck drivers, wanted to improve their customer experience by increasing the face recognition tool accuracy. The tool is used by Lynk to verify truck drivers identity. To achieve this Lynk conducted a Hackathon asking the participants to get an Application programming interface (API), whose functionality is to accept an image as input and give its corresponding person name as output. The datasets provided had all possible computer vision challenges like illumination variations, pose/viewpoint variations, facial expressions, and occlusions. As a conventional procedure, the region which contains face is cropped out first. Image crop procedure is done using openface module provided by Facebook. After getting the face image, training data is constructed. To extract features from the face image, Dense net is used. The advantage of Dense net is, it is the top performer in the ImageNet challenge and it removes the barrier of hand engineered features. The obtained features are classified using fully connected network. The complete setup was made as an API using Django. The network performed better for challenging images. The dataset was very less with only 10-15 images in each class for a total class of 23. Data augmentation is done immensely, the batch size of 16 will have 50% of the actual images and the remaining 50% are augmented images.

Automatic Polyp detection in Colonoscopy videos

Colonoscopy is the primary method for colon cancer screening and prevention, during which a tiny camera is navigated into the colon in order to find and remove polyps(precursors to colon cancer). An automatic polyp detection system can point out the locations of polyps in the videos, increasing the attentiveness of colonoscopists during procedures, helping them reduce their polyp miss-rates. The problem statement can be two-fold: 1. Say whether the image contains polyp or not. 2. Locate the polyp if it is present. The first one falls under classification task. Convolutional neural network(CNN) with different architectures like AlexNet, VGGNet, GoogleNet, ResNet is studied. ResNet and VGGNet performed well on the whole with an accuracy of 96.01% in train data and 93.5% in test data. Dataset is taken from Endovis challenge-2017. Imagenet pre-trained weight is used and transfer learning is done. The second one falls under localization task. Since the accuracy was coming high and to confirm whether the network learned the polyp, the problem is treated as visualizing CNN. An interesting paper on Class activation map + Global Average Pooling is implemented which localizes in the polyp region showing that network did actually learn the polyp.

Sonosight - Augmented Reality based sound visualization wearable

Persons with hearing impairment often rely on visual signals such as gestures and lip movement to interpret speech. While hearing aids can improve sound recognition, they generally do not help the hearing impaired user localize sound which allows them to identify visual cues. In outdoor environments, they lack the common contextual signs like a honking of a horn from behind the user or a person trying to converse when out of the line of sight. SonoSight is an assistive wearable device which attempts to solve this using Augmented Reality and sound localization to identify sound sources and project it in the peripheral vision of the user. SonoSight uses a quad microphone array to identify sound sources. The device through cloud connectivity leverages an RNN based sound source identification to identify the sound that surrounds the user and provides a real-time voice to text captioning for the user.

Efficient tool to process high resolution images.

The major problem with processing high resolution images is their size. Even performing conventional image processing algorithms can be cumbersome. So the easiest way is to split the high resolution image into several tiles of lesser dimensions, process each tile and combine the result in the end. This operation will become even more efficient if the obtained tiles are processed parallely. The Univa grid engine(100 node cluster) provided by the Cold Spring Harbor Laboratory is used to address the problem. An API is developed using python which helps the user to process the high resolution image by providing image as an input. Algorithms can be written in matlab,c++.

Mitosis cell detection - Whole slide Imaging

Breast cancer early diagnosis can be obtained from number of mitotic cells. Pathologists move the slides and look for mitosis cells in the whole slide. The problem with this approach is it is highly arduous and time consuming task. And the other biggest worry is variation of selecting mitotic cells between pathologists. As the views here are subjective. Having an automated mitotic cell detection helps in assisting the pathologists and make their work easier. The detection of mitotic cells is not an easy task because of its similarity with nuclei cells. Since mitosis undergoes four stages, the intra-class variance of the mitotic class is high, while the inter class difference between the mitosis and non-mitosis cell is low. This makes it even more hard to design a classifer. To verify the model, histopath research community has given us three publicly available dataset. 2012,2014 and TUPAC. We have taken multiple approaches for this problem. 1. Treating the mitosis as an object, state of the art object detectors like Faster R-CNN are used. Since the image is of higher resolution, generally the whole images are patches into small segments and each segment is given to a detector. After obtaining the sub-patch coordinates, it will be mapped to the whole slide. Since Faster R-CNN has an in-built classifier the result will be a bounding box coordinates with confidence score. 2. Treating the same thing as a segmentation problem, segmentation of mitotic cells are carried out for the sub-patches. After performing the segmentation for each patch it is mapped back to the whole image. Now this cells are given to state-of-the art classifiers.
